# SIEM / Threat Hunting System with ELK Stack

A SIEM laboratory built using ELK Stack, Elastic Agent, Fleet Server, Kibana, and Elastic Defend to support centralized logging, security monitoring, threat detection, and security event investigation.

## Features

- Centralized log collection and normalization
- Security monitoring and alerting
- Detection rules and alert investigation
- Kibana dashboards and visualizations
- Endpoint monitoring with Elastic Defend
- KQL-based log analysis and investigation
- Machine Learning anomaly detection
- Multi-zone network architecture (DMZ, Internal User, Database, SIEM)

## Security Scenarios

- Port Scanning Detection
- SQL Injection Detection
- Directory Scanning Detection
- EICAR Endpoint Security Testing
- Host Isolation with Elastic Defend
- Machine Learning Anomaly Detection
